== Description ==

This plugin will add a noembed section no any embedded content that does not already have one. This is very handy for people reading your site in an RSS reader, as a lot of them don't display embedded content. 

Also, for any Youtube or GameVideos.com videos, this plugin will also show a thumbnail of the video. The message displayed can be customised from the options page for the plugin.  

See the [Plugin Homepage](http://blog.slaven.net.au/wordpress-plugins/noembedder-wordpress-plugin/) for more information.

== Installation ==

1. Download the zip file & extract the contents.
1. Copy the wp-noembedder folder to the "/wp-content/plugins" folder of WordPress.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. (optional) Modify the 'Noembed Text' in the Noembed Options page
